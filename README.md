# Ano Novo, Corpo Novo — site estático
Conteúdo: site minimalista para "Ano Novo, Corpo Novo: Emagreça com facilidade até o dia 31!"

## Arquivos
- index.html, sobre.html, dicas.html, contato.html
- assets/styles.css
- assets/scripts.js

## Como publicar no GitHub Pages (grátis)
1. Crie uma conta em https://github.com e faça login.
2. Crie um novo repositório. Você pode nomeá-lo `seunome.github.io` para publicar no root, ou qualquer nome (vai ficar em https://usuario.github.io/nome-repo).
3. Faça upload dos arquivos deste zip para a branch `main` (arraste e solte via web UI) — certifique-se que `index.html` está na raiz do repositório.
4. Vá em **Settings > Pages** (ou `Settings > Code and automation > Pages`) e escolha a branch `main` e `/ (root)`. Salve. O site estará disponível em alguns minutos.
5. Teste no celular: abra o link no Android/iOS.

## Como publicar no Netlify (grátis)
1. Crie conta em https://app.netlify.com e faça login.
2. Clique em "Sites" -> "Add new site" -> "Deploy manually" -> arraste o conteúdo do zip ou a pasta descompactada.
3. O Netlify fornecerá um domínio temporário com HTTPS. Você pode trocar para domínio personalizado nas configurações se desejar.

## Observações
- As imagens usadas são links externos (Unsplash). Se quiser que as imagens fiquem locais, substitua os `src` por arquivos em `assets/img/` e atualize o HTML.
- O botão "Compartilhar" copia o link atual para a área de transferência.
